# Project: Novu Research

**Description:** Egy modern kutatói dashboard és platform, ami a DeepMind Science Skills csomagját teszi elérhetővé és vizualizálhatóvá kutatók és biohackerek számára.
**Status:** Planning / Idea Phase
**Key dates:** 2026-05-26: Projekt indítása, jogi és technikai alapozás.

## Core Vision
Az "AI Research OS" megteremtése. Nem csak adatot szolgáltatunk, hanem kontextust és érthető vizualizációt a legmodernebb tudományos skillek köré.

## Jogi státusz (Hardy döntése)
- A licenc (Apache 2.0) engedi a kereskedelmi felhasználást.
- A modell: Az AI által feldolgozott adatot és a platformélményt értékesítjük.

## Technikai Stack (Tervezett)
- **Frontend:** React + Vite + Tailwind + Shadcn UI
- **Backend:** Supabase (auth, database)
- **Engine:** Gemini CLI + Science Skills integration

## Funkciók (Ötletelés alatt)
1. Literature Explorer (PubMed/Arxiv)
2. Protein Visualizer (AlphaFold)
3. Genomic Insight (gnomAD)
