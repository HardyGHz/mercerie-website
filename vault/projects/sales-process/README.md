## Ajánlati logika

**Belépő:** Ingyenes audit (auto service-nél) vagy konkrét fájdalompont azonosítása
**Core:** Website mint fundáció minden egyébhez
**Upsell:** Voice agent, Facebook Ads, SEO, email automatizálás

**Árazási elv:** Értékalapú — a hozott érték 5-15%-a
- Voice agent: ügyfél napi elveszített hívások × átlag megrendelés értéke = havi elveszített bevétel → ebből 5-15%
- Példa dezmembrare: 3 elveszített hívás/nap × 400 lej = ~24.000 lej/hó → agent ára: 300-1.500 lej/hó

## Sales Script (5 mondatos nyitó)
1. Kapcsolódás: "Láttam a munkátokat, van egy ötletem a több ügyfélért."
2. Felütés: "Összeraktam egy prototípust, ami átadja a minőségeteket."
3. Bemutatás: "Mobilra optimalizált automata rendszer a vevőknek."
4. Érték: "0-24 gyűjti a neveket és telefonszámokat, nem csak egy kép."
5. Zárás: "Segítene ez nektek a több megkeresésben?"
